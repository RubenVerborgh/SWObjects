SPARQLfed Source Code Directory

① SPASQL Example · ② SPASQL-MySQL · ③ SPASQL XTech Paper · ④ XTech Slides · ⑤ SPASQL CVS Tree

This is the source for a SPARQL parser and modules to implement a SPASQL engine in MySQL, as well as Rule Mapping functionality.

  • Makefile — targets for libSPARQLfed.a and SPARQLfedTest.
  • SPARQLfedTest.. — simple main to link to libSPARQLfed.a.
  • ParserCommon.cc and ParserCommon.hh — common Driver source for all RDF-related parsers.
  • SPARQLfedParser.yy, SPARQLfedScanner.hh and SPARQLfedScanner.ll — parser for the SPARQLfed language.
  • TurtleParser.yy, TurtleScanner.hh and TurtleScanner.ll — parser for the Turtle language.
  • RdfDB.cc and RdfDB.hh — implementation of a simple RDF database with multiple named graphs.
  • RdfQueryDB.cc and RdfQueryDB.hh — overload of RdfDB which performs queries on graph patterns.
  • ResultSet.cc and ResultSet.hh — simple SPARQL result set data structure.
  • SPARQL.cc and SPARQL.hh — data structures to represent graph patterns for SPARQL and N3.
  • SPARQLDuplicator.hh — deep-copy of a SPARQL compile tree (RDF Atoms allocated from a POSFactory.
  • SPARQLTransformer.hh — overload of SPARQLDuplicator which to mutate the compile tree in interesting ways.
  • XMLSerializer.hh — simple XML representation of a SPARQL query.
